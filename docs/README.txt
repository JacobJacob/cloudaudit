CloudAudit Documentation

To contribute to the documentation you will need:
 - Subversion
 - XMLmind XML Editor (free personal edition is OK)
 - XML2RFC extension for XMLmind (http://code.google.com/p/xml2rfc-xxe/)

The process is simple:
1. Edit .xml file(s)
2. Run 'make' (requires Internet as uses xml.resource.org CGIs rather than local tools)
